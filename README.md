## рҹ§  How It Works

1. Upload a soil image  
2. Flask backend processes the image  
3. Deep learning model predicts soil type  
4. System generates insights:
   - Soil Type  
   - Moisture Level  
   - Soil Strength  
   - Crop Recommendation  
   - Confidence Score  
   - Soil Health Score  
5. Results displayed on dashboard  
6. Stored in SQLite database
